# Zasady Portfela Inwestycyjnego

## Profil

- **Horyzont:** 1–3 lata
- **Kapitał:** 50k–250k PLN
- **Tolerancja ryzyka:** wysoka — szukam 10–50x asymetrii
- **Giełdy:** globalny mandat (US, EU, PL)
- **Sektory moonshot:** AI, Energy (fusion/SMR/nuclear), Space, Quantum, Robots, Health, Defence

## Metodologia wyceny

- Główna metoda: **DCF** (intrinsic value vs. market price)
- Referencja: Morningstar fair value estimate + moat rating
- Obowiązkowe elementy analizy: teza biznesowa, Buffett tenets, model DCF, entry point, ryzyka, czynnik geopolityczny
- **Margin of safety:** minimum 20–30% poniżej fair value przy zakupie

## Zasady zarządzania rekomendacjami

- **Żadna zmiana `recommendation`, `entry_point` ani `thesis_breaker` bez jawnej akceptacji** ("tak" / "zmień" / "akceptuję")
- Asystent może proponować zmiany — nie może ich samodzielnie aplikować
- Rekomendacja zawsze zawiera: entry point + thesis breaker (kiedy wyjść)

## Watchlista — zasady

- Spółka kupiona → przenieść do portfela, usunąć z watchlisty
- Spółka sprzedana → zostaje na watchliście z notatką o re-entry
- Re-entry: nowy entry point + konkretny trigger przed zakupem
